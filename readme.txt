The best way to share your trending topics directly from your website.

== Description ==

Have a Twylah page? Add the Twylah widget to your website!

* Showcase your trending topics
* Display thumbnail images for visual appeal and engagement
* Link directly to your Twylah page

Note: You must have an existing Twylah page to install the Twylah widget to your Wordpress site. You can sign up for a Twylah page at [www.twylah.com](http://www.twylah.com/)

== Installation ==

Installation is simple.

1. Unzip the downloaded file and upload the folder inside to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In the WordPress Admin, navigate to Appearance->Widgets.
4. Add the Twylah Wordpress widget by dragging into the sidebar.
