Share your trending topics directly on your website.

== Description ==

Twylah organizes your tweets into trending topics showcasing your brand message. Now you can add the Twylah widget directly on your website!

* Showcase your trending topics.
* Display thumbnail images for visual appeal and engagement.
* Link directly to your Twylah page.

Note: You must have an existing Twylah page to install the Twylah widget to your Wordpress site. You can sign up for a Twylah page at [www.twylah.com](http://www.twylah.com/)

== Installation ==

Installation is simple.

1. Unzip the downloaded file and upload the folder inside to the /wp-content/plugins/ directory or use the wordpress plugin installer in 'Add New' then search for 'Twylah'.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. A new 'Twylah Wordpress Widget' will be available under Appearance > Widgets.
4. Add it to your sidebar and edit settings of the plugin.
